## Project
rawToDICOM-py — Pipeline for processing raw, cardiac MRI data into DICOM files.
Translated from a MATLAB legacy pipeline into Python. See README.md for full documentation.

## Stack
- Python 3.10
- MATLAB R2025b (legacy code in MATLAB-legacy/)

## Structure
- MATLAB-legacy/ — legacy code (readBrukerParamFile.m, readBrukerRaw.m, RawDataObject.m are the Bruker I/O layer)
- src/rawtoDICOM/ — Python package
  - config.py     — PipelineConfig frozen dataclass (replaces MATLAB pm struct)
  - cli.py        — CLI entry point (`python -m rawtoDICOM`); flat CLI, auto-detects SG vs CINE
  - bruker/       — Bruker file I/O: scan.py (BrukerScan dataclass), params.py (JCAMP-DX parser), raw.py (binary reader), reader.py (load_scan / scan_plane / is_sg_scan)
  - reconstruction/ — kspace.py (fft2c/ifft2c/sort_kspace/zero_fill_kspace), compressed_sensing.py (CSConfig/reconstruct_cs), coil_combination.py (combine_coils)
  - reconstruction/self_gating/ — reader.py, navigator.py, peak_finder.py, shuffler.py, synchronizer.py
  - dicom/        — affine.py (BrkRaw affine math + bruker_to_lps + orient_correction_brkraw), geometry.py (apply_corrections, shuffle_slices), writer.py
  - pipeline/     — sort.py (sort_raw_data), cine.py (process_cine_scan), self_gating.py (process_sg_scan)
- tests/ - test files (mirrors src/ structure)
- tests/raw-data/ - real Bruker ParaVision test data; structure mirrors production: raw-data/{project}/{cohort}/{subject}/{scan_number}/
- tests/examples/ - animated GIFs of pipeline stages and LAX/SAX co-registration (generated by `pytest -m slow`)
- tests/sorted-data - output of sort_raw_data (empty until pipeline runs)
- tests/DICOM-data - DICOM output (empty until pipeline runs)

## Key architecture decisions
- BrukerScan is a frozen dataclass — no mutation after construction
- One reconstruct_cs() covers both CINE and self-gating (two MATLAB versions unified)
- No intermediate .mat file saves — arrays flow directly between pipeline steps
- CINE and SG pipelines share combine_coils() and reconstruct_cs()
- Self-gating is a sub-module of reconstruction/, not a separate top-level tree
- Pipeline auto-detects SG vs CINE per scan via is_sg_scan(); no manual dispatch needed
